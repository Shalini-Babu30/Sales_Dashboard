Sales Performance Dashboard

Project Overview
This project is an interactive dashboard created as part of a Data Analyst Internship Task. The dashboard provides insights into sales performance, profit trends, and regional distribution using Power BI.

Objectives
* Analyze sales and profit data
* Identify trends over time
* Compare performance across categories
* Visualize regional/state-wise sales distribution
* Create an interactive dashboard for business decision-making

Dataset
* Sales Financial Dataset (~1000 records)
* Includes:
  * Sales (Amount)
  * Profit
  * Order ID
  * Category & Sub-category
  * State
  * Payment Mode
  * Date

Dashboard Features
*  KPI Cards:
* Total Sales
* Total Profit
* Total Orders
* Time-Series Analysis:
   * Sales trend over time using line chart
Comparison:
   * Profit by Category using bar chart
* Distribution:
    * Sales by Top States using pie chart
* Interactive Filters (Slicers):
  * Category
  * Sub Category
  * Payment Mode

Key Insights
* Sales showed growth until 2024 and declined in 2025
* Office Supplies category generated the highest profit
* A few top states contribute the majority of total sales
* Payment mode impacts purchasing patterns

Tools Used
* Power BI (Dashboard creation)
* Excel (Data cleaning)
* GitHub (Project submission)

Conclusion
This dashboard helps in understanding business performance, identifying trends, and making data-driven decisions using interactive visualizations.
